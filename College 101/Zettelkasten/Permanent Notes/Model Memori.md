# Model Memori

Model memori (Atkinson & Shiffrin, 1968) menjelaskan bahwa informasi melewati tiga sistem penyimpanan secara berurutan sebelum dapat disimpan secara permanen. Model ini sering disebut sebagai model memori tiga-sistem (three-stage model of memory).

| Sistem | Penjelasan | Kapasitas | Durasi |
|---|---|---|---|
| **Sensory Memory** (Memori Sensoris) | Tahap awal pemrosesan; informasi dari lingkungan diterima melalui pancaindra; **perhatian** (attention) menentukan apakah informasi diproses lebih lanjut atau hilang | Sangat besar (semua input sensoris) | Sangat singkat (kurang dari 1 detik untuk visual, beberapa detik untuk auditori) |
| **Working Memory** (Memori Kerja) | Pusat aktivitas berpikir manusia; informasi yang mendapat perhatian diproses secara aktif; tempat menghubungkan informasi baru dengan pengetahuan yang sudah ada | Terbatas (sekitar 7 plus/minus 2 item) | Singkat (15-30 detik tanpa rehearsal) |
| **Long-Term Memory** (Memori Jangka Panjang) | Tempat penyimpanan informasi secara relatif permanen | Sangat besar, nyaris tidak terbatas | Relatif permanen |

Alur pemrosesan: **Sensory Memory** => **Working Memory** => **Long-Term Memory**

Tulving membedakan dua jenis long-term memory:
- **Memori episodik** (episodic memory) => penyimpanan pengalaman pribadi yang terikat waktu dan tempat (contoh: kenangan liburan keluarga)
- **Memori semantik** (semantic memory) => penyimpanan fakta, konsep, dan pengetahuan umum yang tidak terikat konteks spesifik (contoh: mengetahui bahwa Jakarta adalah ibu kota Indonesia)

- **Perhatian** berperan krusial sebagai "filter" yang menentukan informasi mana dari sensory memory yang masuk ke working memory
- Informasi di working memory perlu diproses secara aktif (encoding) agar dapat dipindahkan ke long-term memory
