# Token Economy

**Token Economy** (Ekonomi Token) adalah sistem penguatan terstruktur di mana subjek mendapatkan token setiap kali melakukan perilaku target, dan token tersebut dapat ditukarkan dengan backup reinforcer (penguatan cadangan) yang bernilai nyata bagi subjek.

## Komponen Token Economy

Token economy berfungsi dengan baik ketika keempat komponen berikut didefinisikan dengan jelas:

1. **Perilaku target**: perilaku spesifik apa yang akan menghasilkan token
2. **Token**: bentuk fisik atau simbolis yang diberikan (stiker, poin, chip)
3. **Backup reinforcer**: reward nyata yang dapat ditukar dengan token
4. **Exchange rate**: berapa token dibutuhkan untuk mendapat backup reinforcer tertentu

## Mengapa Token Economy Efektif

Token berfungsi sebagai [[Penguatan Terkondisi]]: awalnya netral, tetapi memperoleh nilai penguat karena konsisten dipasangkan dengan backup reinforcer. Karena token tidak kenyang (tidak seperti makanan atau minuman), token dapat digunakan berulang kali tanpa kehilangan kekuatannya sebagai reinforcer.

Selain itu, sistem token menjembatani jarak waktu antara perilaku dan reward, sehingga cocok untuk situasi di mana backup reinforcer tidak bisa diberikan langsung saat perilaku muncul.

## Keterbatasan

Token economy membutuhkan konsistensi tinggi dari pihak yang mengelola program. Jika token tidak diberikan secara konsisten mengikuti aturan yang sudah ditetapkan, sistem akan kehilangan efektivitasnya.
