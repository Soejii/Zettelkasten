# Working Memory

**Working Memory** (Memori Kerja) adalah sistem kognitif yang secara aktif menyimpan dan memanipulasi informasi dalam jangka pendek selama proses berpikir, belajar, memahami, dan penalaran berlangsung. Ia bukan sekadar "tempat penyimpanan sementara," tetapi ruang kerja aktif tempat informasi diolah secara sadar.

Dalam [[Model Memori]] Atkinson & Shiffrin (1968), working memory disebutkan sebagai tahap kedua dalam alur pemrosesan informasi, dengan kapasitas sekitar **7 ± 2 item** (Miller, 1956) dan durasi **15-30 detik** tanpa rehearsal.

## Model Baddeley (1974)

Baddeley & Hitch mengusulkan bahwa working memory bukan unit tunggal, melainkan terdiri dari beberapa komponen:

| Komponen | Fungsi |
|---|---|
| **Central Executive** | Kontroler utama; mengatur perhatian, mengoordinasikan komponen lain, dan mengintegrasikan informasi dari berbagai sumber |
| **Phonological Loop** | Menyimpan dan melatih informasi berbasis suara/verbal (inner voice); bertanggung jawab atas kemampuan mengingat urutan kata atau angka |
| **Visuospatial Sketchpad** | Menyimpan dan memanipulasi informasi visual dan spasial; digunakan saat membayangkan peta, rotasi objek, atau denah |
| **Episodic Buffer** | Komponen integrasi yang menghubungkan informasi dari phonological loop, visuospatial sketchpad, dan long-term memory menjadi representasi terpadu (ditambahkan Baddeley tahun 2000) |

## Kapasitas dan Chunking

Kapasitas working memory terbatas. Miller (1956) menyebutnya "the magical number 7 ± 2," sementara penelitian lebih mutakhir (Cowan, 2001) memperkirakan kapasitas sebenarnya lebih dekat ke **4 ± 1 item**. Keterbatasan ini dapat disiasati melalui **chunking**, yaitu mengelompokkan beberapa item menjadi satu unit bermakna yang diperlakukan sebagai satu "slot" dalam working memory (contoh: mengingat "081234567" sebagai "0812-345-67" bukan sembilan angka terpisah).

## Peran Perhatian

[[Perhatian (Attention)]] berperan sebagai gerbang masuk ke working memory. Informasi dari sensory memory hanya akan masuk ke working memory jika mendapat perhatian (selective attention). Central executive dalam model Baddeley juga berfungsi sebagai pengatur perhatian, menentukan sumber daya kognitif mana yang diaktifkan.

## Relevansi dalam Belajar

Working memory yang terbatas adalah alasan mengapa beban kognitif (cognitive load) dalam pembelajaran perlu dikelola. Materi yang terlalu kompleks atau disajikan terlalu cepat akan melebihi kapasitas working memory, menghambat transfer informasi ke long-term memory.
